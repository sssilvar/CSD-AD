# Config folder
Here you put all your configuration files including relevant information as follows:

<code>
[dirs] <br>
    dataset_folder: ...
    dataset_folder_registered: ...
    subjects_dir: ...
    sphere_mapping: ...

# You can also declare variables
[params]
    home=/home/username
    documents: %(home)s/Documents

[resources]
    n_cores: ...
</code>
